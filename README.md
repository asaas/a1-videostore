# Q1 VideoStore #

## 課題

このプロジェクトをテストコードも含めてリファクタリングしてください。
JDK 8 でコンパイルできるコードを提出してください。

## 実行

Maven をインストールした状態であれば、テストコードは下記のコマンドで実行できます。

```
$ mvn test
```

## 提出

GitHub フローにしたがってフィーチャーブランチをプッシュしてプルリクエストを送信してください。
コミットをスカッシュする必要はありません。
